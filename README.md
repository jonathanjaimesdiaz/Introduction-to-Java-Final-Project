# Introduction-to-Java-Final-Project

A custom Java application built on NetBeans the final project for CIT 130 (Beginning Java) at TMCC, Spring 2026.

## What It Does
The BlackJack Casino program is a menu-based casino game where the user can create a player, enter starting money, place bets, and play Blackjack against a computer dealer. The game uses object-oriented programming concepts such as classes, objects, methods, arrays, loops, and exception handling while allowing the player to hit, stand, double down, win or lose money, and continue playing until they run out of money or choose to quit. 

## Concepts Demonstrated
- Multiple classes with overloaded and default constructors
- Single and multidimensional arrays
- Control flow: for, while, do/while, and enhanced for loops
- Selection: if/else and switch statements
- Static variables
- Exception handling
- Methods that accept and return values
- Proper naming conventions (camelCase, UpperCamelCase, ALL_CAPS)
